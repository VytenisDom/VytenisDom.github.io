About the Project

This Personal website was made with intension to make it easier for future employers to quickly review my portfolio projects, to make it faster and easier for people to get to know me and also for my own pleasure. This website was built from scratch using HTML5, SASS and Javascript mixed with some jQuery. In this project I wanted to express myself and build a website designed in a way that I would like it to be, so I tried to use various modern design trends while keeping it rather non-flashy and down to earth.
