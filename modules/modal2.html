About the Project

This is the content management system ( CMS ) for the E-commerce shop project that I have done. This was mostly back-end, since I took a HTML5 template and started writting code in PhP to link the two up. However, I still had to style the forms individually, so there were some fiddling around with HTML and CSS. Also some vanilla JS and jQuery for data validations. This CMS is definetly not that complex, the main objectives were to make it fast and easy to use.
