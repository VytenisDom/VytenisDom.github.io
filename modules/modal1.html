About the Project

This project is my biggest one yet, mainly showcasing my back-end skills. It started off as a yearly project in the programming school I went to. We were learning PhP so we had to take an HTML5 template of an e-commerce shop and started working. Even though most of my time was spent on the back-end, there were countless bugs in the template, that really slowed down the development. Nonetheless, the project is mostly finished. Now it's being used for a family business, and already has some traffic going through it.
