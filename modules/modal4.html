Toorist

Completely Hands-off Audio Guide Of Vilnius

About the Project

Me and my team started developing Toorist in a local hackathon called Vilnius Open Data Hackathon 2019. The project is finally finished and consists of an Android App, Ios App, Node.Js server, MySql database and an Admin Panel website. I've been responsable for server logic and requests, and was coordinating the team at the hackathon. This project in it's alpha stage had won 2 prizes of the event, and is on route to be published to the public. Within the app, the user can specify what kind of tourist attractions he prefers, and our system generates a route, according to his preffered categories and starting from where the user is now. More info about the project can be found in our charter here.
